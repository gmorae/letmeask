# :construction_worker: How to run
```bash
# Clone Repository
$ git clone https://github.com/gmorae/letmeask.git

# Go to server folder
$ cd letmeask

# Install Dependencies
$ yarn or npm i

# Run Aplication
$ yarn start or npm start
```

Go to http://localhost:3000/ to see the result.
